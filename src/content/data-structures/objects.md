unordered key value pairs

#### When to use objects?

- when you need fast access, insertion and removal
- when order doesn't matter

#### Big O of common operations

| Operation | Big O |
| :-------- | :---: |
| insertion | O(1)  |
| removal   | O(1)  |
| access    | O(1)  |

#### Big O of common methods (javascript)

| Operation      | Big O |
| :------------- | :---: |
| Object.keys    | O(n)  |
| Object.values  | O(n)  |
| Object.entries | O(n)  |
| hasOwnProperty | O(1)  |
