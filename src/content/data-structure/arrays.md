#### Big-O of common operations

| Operation                          | Big O |
| :--------------------------------- | :---: |
| access                             | O(1)  |
| searching                          | O(n)  |
| insertion/removal at the end       | O(1)  |
| insertion/removal at the beginning | O(n)  |

#### Big O of common methods (javascript)

| Method            |    Big O    |
| :---------------- | :---------: |
| push              |    O(1)     |
| pop               |    O(1)     |
| forEach           |    O(n)     |
| map/filter/reduce |    O(n)     |
| sort              | O(n log(n)) |
| shift             |    O(n)     |
| unshift           |    O(n)     |
| slice             |    O(n)     |
